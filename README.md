# AI Scholarship & Career Mentor 🤖🎓

## 📌 Project Overview
This is an AI-powered Career Mentor system built using Python and Google Gemini API.
It helps students with career guidance including resume improvement, scholarships, interview preparation, and career roadmap.

## 🚀 Features
- Resume Guidance
- Scholarship Guidance
- Interview Preparation
- Career Roadmap
- AI Chat Assistant (Gemini Powered)

## 🧠 Tech Stack
- Python
- Google Gemini API
- dotenv

## ⚙️ How to Run

```bash
pip install google-genai python-dotenv
python main.py
