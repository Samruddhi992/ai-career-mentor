AI Scholarship & Career Mentor 🤖🎓

📖 Overview

AI Scholarship & Career Mentor is a Python-based intelligent assistant designed to help students make informed career decisions. The system provides personalized guidance for resume building, scholarship opportunities, interview preparation, and career planning using Google's Gemini AI.

🎯 Objectives

- Help students improve their resumes.
- Provide scholarship recommendations and guidance.
- Assist with interview preparation.
- Generate career roadmaps based on user interests.
- Deliver AI-powered career mentoring through natural language interaction.

✨ Features

📄 Resume Guidance

- Resume improvement suggestions
- Skill recommendations
- Project ideas for students

🎓 Scholarship Guidance

⚙️ Installation

Clone the Repository

git clone https://github.com/Samruddhi992/ai-career-mentor.git
cd ai-career-mentor

Install Dependencies

pip install google-genai python-dotenv streamlit

Configure Environment Variables

Create a ".env" file and add:

GEMINI_API_KEY=YOUR_API_KEY

▶️ Run the Application

Command Line Version

python main.py

Streamlit Version

streamlit run app.py
